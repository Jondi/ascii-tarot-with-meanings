A CLI tool for tarot reading with full ASCII art deck \*:･ﾟ✧\*:･ﾟ✧

<img src='ascii-tarot.png' />

## Installation

`npm install -g ascii-tarot`

## Usage

`$ ascii-tarot`

Choose one card, three cards, or celtic cross. Reversed cards are possible.
